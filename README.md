# Shellsea Nunez-Aviles's Portfolio project

A personal portfolio web app built with React Native Web and Expo. Showcases projects, skills, and contact information!

---

## Features

- **Sticky navbar** with active section highlighting that updates as you scroll
- **Hero section**: split layout with text on the left and a full image on the right (desktop), image background on mobile
- **About section**: photo alongside bio, interests, and skills
- **Projects grid**: 2-column responsive grid with live demo and source code links
- **Contact form**: client-side validation with social links
- Fully responsive for both desktop and mobile

---

## Setup

**Prerequisites:** Node.js and npm installed.

1. Clone the repository:
   ```bash
   git clone https://github.com/aesheeds/your-repo-name.git
   cd portfolio-project
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npx expo start
   ```

4. Open in browser:
   - Press `w` in the terminal, or visit `http://localhost:8081`
